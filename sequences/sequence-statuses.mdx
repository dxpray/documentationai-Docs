---
title: Sequence Statuses
description: What each sequence status means — from Draft through In Progress to Completed.
---

Every sequence carries a status, shown as a pill on the Sequences list and the sequence detail page.

| Status | What it means |
| :---- | :---- |
| **Draft** | Created but not yet launched. You can still edit all details. |
| **Preparing** | Getting ready to send — Rose is setting up the outreach. |
| **In Progress** | Live. Candidates are receiving emails. |
| **Paused** | Temporarily on hold; can be resumed. |
| **Completed** | All candidates have received the full sequence. Rose still notifies you if a candidate replies later. |
| **Cancelled** | Stopped by you. Cancelled sequences are tucked away under a separate section on the list. |

<Callout kind="warning">
  Cancelling a sequence after it has launched does not refund credits already used on contacted candidates. See [Billing & Subscription](/billing-and-subscription/credits).
</Callout>
